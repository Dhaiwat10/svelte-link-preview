# Svelte Link Preview

Svelte port of [react-link-preview](https://github.com/dhaiwat10/react-link-preview). WIP.

## How to use

```bash
yarn add svelte-link-preview
```

```svelte
import { LinkPreview } from 'svelte-link-preview';

<LinkPreview url='https://github.com/dhaiwat10'>
```
